## Why Postgres is Awesome

* Unlimited database size
* 32TB maximum table size
* Cross-platform (*nix, Windows, OSX, FreeBSD)
* Open-source
* Structured Query Language (SQL) compliant
* Supports full text search
* Performant
* Proven
* Scaleable
