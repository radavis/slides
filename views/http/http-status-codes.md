## HTTP Status Codes

* 1xx - Informative
* 2xx - Success
  - 200 - OK
  - 201 - Created
  - 202 - Accepted
* 3xx - Redirect
  - 301 - Moved Permanently
  - 302 - Found
  - 303 - See Other
* 4xx - Client Error
  - 400 - Bad Request
  - 401 - Unauthorized
  - 403 - Forbidden
  - 404 - Not Found
  - 418 - I'm a Teapot (!?!?!)
* 5xx - Server Error
  - 500 - Internal Server Error
  - 503 - Service Unavailable
