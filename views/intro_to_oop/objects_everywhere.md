### Everything in Ruby is an Object!

```
[1] pry(main)> array = []
=> []
[2] pry(main)> array.class
=> Array
[3] pry(main)> Array.superclass
=> Object
[4] pry(main)> hash = {}
=> {}
[5] pry(main)> hash.class
=> Hash
[6] pry(main)> Hash.superclass
=> Object
```
